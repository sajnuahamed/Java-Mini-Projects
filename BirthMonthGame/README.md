\# Birth Month Game



A fun console-based Java program that prompts the user to enter their birth month and displays a humorous, fictional personality description based on the input. The program uses Java's enhanced `switch` expression to match the entered month and provides a unique response for each one.



\## Features



\* Accepts the user's birth month as input

\* Handles input regardless of letter case

\* Displays a unique humorous message for each month

\* Includes a default message for invalid month names

\* Uses Java's enhanced `switch` expression

\* Implements `try-with-resources` for proper resource management



\## Technologies Used



\* Java

\* `Scanner` for user input

\* Enhanced `switch` expressions



\## Sample Output



```text

Enter your birth month: March

People can usually find you trying to befriend a stray pigeon or squirrel on your lunch break.

```



\### Invalid Input



```text

Enter your birth month: Hello



I didn't realize HELLO was a month! I'll have to mark my calendar for the 32nd of HELLO.

```
