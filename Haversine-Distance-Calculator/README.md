Haversine Distance Calculator 🌍

A simple console-based Java application that calculates the great-circle distance between two geographic locations using the Haversine formula.

This java project makes use of the following concepts:
Scanner (User Input)
Math Class (Math.sin, Math.cos, Math.asin, Math.sqrt, Math.pow, Math.toRadians)

Test case 
 New York City (40.7128° N, 74.0060° W) and London (51.5074° N, 0.1278° W) distance ~ 5570 km.


Sample Input
Latitude 1 : 40.7128
Longitude 1: -74.0060

Latitude 2 : 51.5074
Longitude 2: -0.1278
Sample Output
=== Haversine Distance Calculator ===

Distance between the two locations: 5570 km
