# Java Utility Suite ☕

Welcome to my Java learning repository!

This repository contains a collection of mini projects built using **Core Java**. Each project is created to strengthen my understanding of programming fundamentals through practical, hands-on development.

## Objectives

* Practice Core Java concepts# Java Utility Suite

A collection of Java console applications developed while revising Core Java concepts. Each project focuses on practicing a specific programming concept such as user input, control flow, loops, arrays, methods, random number generation, and problem-solving. Together, these projects form a solid foundation before moving on to Object-Oriented Programming and backend development.

## Projects

| Project                       | Description                                                                            |
| ----------------------------- | -------------------------------------------------------------------------------------- |
| Calculator                    | Performs basic arithmetic operations.                                                  |
| Compound Interest Calculator  | Calculates compound interest using user-provided values.                               |
| Haversine Distance Calculator | Computes the great-circle distance between two geographical coordinates.               |
| Temperature Converter         | Converts temperatures between Celsius and Fahrenheit.                                  |
| Number Guessing Game          | A random number guessing game with user feedback.                                      |
| Birth Month Game              | Displays humorous personality descriptions based on the user's birth month.            |
| Matrix Using For Loop         | Converts a one-dimensional array into a two-dimensional matrix representation.         |
| Theater Seating Manager       | Simulates theater seat reservations using a 2D boolean array.                          |
| Dice Rolling Game             | Simulates rolling one or more dice and displays ASCII dice faces with the total score. |
